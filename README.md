## Tips
### 1. How to store Git credential for HTTPS clone
Run
```
git config --global credential.helper store
```
then
```
git pull <or push>
```
**REF** : https://stackoverflow.com/questions/35942754/how-to-save-username-and-password-in-git-gitextension
